# Alexa Skill for Netatmo Healthy Home Coach

German Alexa Skill for Netatmo Healthy Home Coach

## Intents:
- Temperatur
- C02-Gehalt
- Luftfeuchtigkeit
- Lautstärke
- Luftdruck
- Gesundsheitsindex
- Bericht (alle Messwerte)
- Zeitpunkt der letzten Messung
- Device Standort
- Nutzer Emailadresse
- Liste der verknüpften Geräte
- WiFi Status
